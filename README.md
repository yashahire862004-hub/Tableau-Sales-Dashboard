# Comprehensive Car Sales Analysis Dashboard

This repository contains a **Tableau dashboard** project that analyzes car sales data to reveal insights like trends, regional performance, demographics effects, model performance, and more.

---

# Project Overview

The goal of this project is to create a **visual and interactive dashboard** using Tableau. It provides insights into:

- Monthly sales trends
- Sales by car color
- Regional performance (based on sales amount)
- Sales distribution by gender
- Model-wise comparison of sales
- Income segment analysis

# How to View the Dashboard

To open and view the dashboard:

1. Download **Tableau Desktop** or **Tableau Public**
2. Open the file: [Sales Dataset.csv](https://github.com/user-attachments/files/25009326/Sales.Dataset.csv)
3. Interact with the filters and charts

# Dashboard Features

**Top KPI Cards**  
• Total sales  
• Average car price  
• Sales quantity  
• Auto vs Manual distribution  

**Trends & Breakdown**  
• Monthly sales quantity trend  
• Car color distribution  
• Regional sales breakdown on map  

**Segment Analysis**  
• Sales by annual income and gender  
• Model wise sales comparison
